# 简历信息提取智能体 (resume_extractor)

## 概述
这是大学生职业规划智能体平台的**第一个智能体**，负责从多种来源提取用户简历信息，并进行结构化处理。

## 核心功能

### 1. 多源信息输入
支持三种方式采集用户简历信息：

| 输入方式 | 说明 |
|---------|------|
| **手动输入** | 用户直接在文本框中输入自我介绍内容 |
| **文档上传** | 支持 PDF、Word (.doc/.docx) 格式的简历文件 |
| **图片上传** | 支持图片格式，通过 OCR 识别文字 |

### 2. 信息整合
将来自不同来源（文本、文档、图片）的简历信息进行：
- 去重处理
- 合并同类项
- 补全细节
- 按逻辑顺序整理（基本信息 → 教育背景 → 工作/实习经历 → 技能证书 → 自我评价）

### 3. 结构化信息提取
从整合后的文本中提取 7 个维度的结构化信息：

| 维度 | 说明 |
|------|------|
| 专业技能 | 技术、工具、语言、框架等及熟练程度 |
| 证书 | 已获得的证书 |
| 创新能力 | 竞赛获奖、专利、论文等 |
| 学习能力 | GPA、排名、自学成果等 |
| 抗压能力 | 多项目并行、高强度工作等 |
| 沟通能力 | 学生干部、组织活动等 |
| 实习能力 | 实习/项目实践经验 |

### 4. 信息完整度检查
- 检查 7 个维度是否都已填写
- 标记缺失或模糊的维度
- 支持追问补全机制

### 5. 智能追问
当信息不完整时，自动生成追问问题引导用户补充：
- 针对"未提及"的维度直接询问
- 针对"表述模糊"的维度要求具体举例

### 6. 职业能力分析报告
信息完整后，自动生成职业能力分析报告，包含：
- **技能概览**：用户掌握的核心技能总结
- **优势分析**：最突出的能力优势
- **待提升**：需要加强的能力领域
- **职业倾向**：推测可能适合的职业方向
- **发展建议**：针对性的职业发展建议

## 工作流程

```
用户输入/上传
    ↓
┌─────────────────────────────────────┐
│  1. process_text      处理文本输入   │
│  2. process_file      处理文档      │
│  3. process_image     处理图片OCR   │
│  4. integrate_info    整合所有信息   │
│  5. extract_params    提取结构化信息 │
│  6. check_completeness 检查完整度   │
└─────────────────────────────────────┘
    ↓
┌──────────────────┐
│  有缺失?           │
├──────────────────┤
│  是 → generate_question 生成追问  │
│  否 → analyze_profile 生成报告   │
└──────────────────┘
```

## API 接口

### POST /api/start
启动简历信息提取流程

**请求参数（FormData）：**
- `input_text`: 手动输入的文本
- `doc`: 简历文件（PDF/Word）
- `image`: 图片文件

**返回：**
```json
{
  "step": "extract|complete",
  "profile": {...},
  "missing_fields": [...],
  "next_question": "...",
  "analysis": {...}
}
```

### POST /api/supplement
补充缺失信息

**请求参数（JSON）：**
- `supplement_text`: 补充的文本
- `profile`: 当前画像
- `merged_text`: 合并后的文本
- `supplement_count`: 补充次数

## 文件结构

```
resume_extractor/
├── __init__.py      # 模块导出
├── agent.py         # 核心逻辑（6个处理函数 + 工作流）
├── prompts.py       # 4个提示词模板
├── state.py         # 状态定义
└── promotes.md      # 提示词文档
```

## 提示词模块

| 提示词 | 用途 |
|--------|------|
| INTEGRATE_PROMPT | 信息整合 |
| EXTRACT_PROMPT | 结构化提取 |
| ANALYZE_PROMPT | 职业能力分析 |
| QUESTION_PROMPT | 生成追问问题 |

## 技术栈

- **框架**: LangChain + Flask
- **LLM**: DeepSeek Chat
- **OCR**: Tesseract
- **文档解析**: PyPDFLoader, python-docx
