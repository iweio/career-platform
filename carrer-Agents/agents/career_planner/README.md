# 职业规划智能体

## 功能

- 根据用户收藏的最佳匹配岗位，预测该岗位未来5年（2026-2030）的**薪资趋势**和**需求趋势**
- 结合用户画像和晋升通道数据，生成**分阶段职业发展路径**
- 生成可视化趋势图表

## 工作逻辑

```
用户ID → 获取最佳匹配岗位 → 预测薪资/需求趋势 → 生成职业发展路径 → 保存结果
```

### 详细流程

1. **获取最佳匹配岗位**
   - 从数据库 `user_favorites` 表中查询用户收藏的岗位
   - 根据人岗匹配分数排序，获取分数最高的岗位
   - 返回岗位名称、匹配分数等信息

2. **预测薪资与需求趋势**
   - 调用 LLM (DeepSeek) 分析
   - 输入：岗位名称、用户画像（专业技能、实习经验、教育背景等）
   - 输出：2026-2030年的薪资和需求指数预测
   - 使用预定义的提示词模板确保输出格式稳定

3. **生成职业发展路径**
   - 从数据库 `promotion_transition` 表查询该岗位的晋升通道数据
   - 调用 LLM 结合用户画像和晋升数据生成3阶段发展规划
   - 阶段划分：初阶 → 中阶 → 高阶/专家
   - 每个阶段包含：目标、能力要求、薪资预期、时间节点

4. **保存与可视化**
   - 将结果保存到 `career_planning` 表
   - 生成双Y轴趋势图（薪资+需求），保存为PNG

## 数据来源

| 表名 | 用途 |
|------|------|
| `user_favorites` | 获取用户收藏的最佳匹配岗位 |
| `promotion_transition` | 岗位晋升通道数据（当前岗位→下一岗位→所需技能） |
| `career_planning` | 保存职业规划结果 |
